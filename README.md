# 🤖 Bootcamp DIOS - Inteligência Artificial

## 🚀 Sobre o Projeto
<p>
Este projeto foi desenvolvido durante o <strong>Bootcamp da DIO</strong>,
com foco em <strong>Inteligência Artificial</strong> e aplicação prática de conceitos
modernos utilizando JavaScript.
</p>

<p>
O objetivo principal é explorar funcionalidades relacionadas à IA,
automação e lógica de programação, aplicando boas práticas de desenvolvimento.
</p>

## ⚙️ Funcionalidades
<ul>
  <li>Implementação de lógica com IA</li>
  <li>Organização modular do código</li>
  <li>Execução de scripts automatizados</li>
  <li>Manipulação de dados</li>
</ul>

## 🧠 Tecnologias Utilizadas
